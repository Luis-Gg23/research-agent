# Research Agent with Persistent Memory

An AI-powered research agent that searches the web, summarizes findings,
and remembers context across sessions using RAG and vector storage.

## What it does

- Searches the web for information on any given topic using Tavily
- Summarizes and extracts key insights from search results
- Stores findings in a vector database (ChromaDB) for persistent memory
- Answers questions using both its base knowledge and past research sessions
- Simple chat interface built with Streamlit

## Tech Stack

- **LangChain** — agent orchestration and tool calling
- **OpenAI GPT-4o** — language model
- **Tavily** — real-time web search
- **ChromaDB** — vector storage for persistent memory
- **Streamlit** — chat interface
